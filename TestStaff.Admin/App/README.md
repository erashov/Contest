# Интерфейс администратора для тестирования персонала

## Возможности
- Создание теста
- Добавление разделов
- Добавление вопросов к тесту
- Добавление ответов к вопросу
- Добавление клиентов
- Создание группы рассылки
- Отправление опроса группе рассылки
- Выгрузка отчета

## Сборка
> npm run build
